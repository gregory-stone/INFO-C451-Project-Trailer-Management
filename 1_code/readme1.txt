Trailer Management is meant to run on a local server

Setup
1. Save the contents of the 1_code directory to a local directory C:\xampp2\htdocs\TrailerManagement
2. Using XAMPP, start the Apache and MySQL modules
3. Create and populate the initial MySQL database tables and sample data by executing the queries in the trailer_management.sql file

Use
1. open a browser and navigate to http://localhost:8081/TrailerManagement/Index.php
2. Login with valid credentials. 
	System administrator user:
		username: admin
		password: admin

3. The navigation links on the left side give access to read various data.
4. The settings link gives access to user setup, location setup, and company/warehouse setup.

Users will have different read/write settings to view/add/update/delete records in the various tables.
